## 摘要

本文围绕 Gymnasium 中的 LunarLander-v3 离散控制任务，设计并实现了基于深度 Q 网络的强化学习智能体。项目首先完成本地 Python 实验环境搭建和 LunarLander 环境验证，然后实现基础 DQN，包括 Q 网络、经验回放池、epsilon-greedy 探索策略、target network 和训练循环。在此基础上，后续引入 Double DQN、Dueling Network 和 Prioritized Experience Replay 等改进方法，通过训练曲线、平均奖励、成功率和推理效果对不同方法进行对比分析。实验结果用于验证改进方法对训练稳定性、收敛速度和最终控制效果的影响。

关键词：强化学习；DQN；Double DQN；Dueling Network；优先经验回放；LunarLander

## 2 相关理论

### 2.1 强化学习基本概念

解释以下概念：

- Agent：智能体
- Environment：环境
- State：状态
- Action：动作
- Reward：奖励
- Episode：一局完整交互过程

结合本项目说明：

- 状态是 LunarLander 的 8 个观测值。
- 动作是 4 个离散发动机控制动作。
- 奖励由环境根据着陆位置、速度、角度、是否坠毁等因素给出。

### 2.2 马尔可夫决策过程

说明强化学习任务可以建模为 MDP：

```text
(S, A, P, R, gamma)
```

其中 `S` 是状态空间，`A` 是动作空间，`P` 是状态转移概率，`R` 是奖励函数，`gamma` 是折扣因子。

### 2.3 Q-learning 与 DQN

说明 Q-learning 学习的是动作价值函数：

```text
Q(s, a)
```

含义是在状态 `s` 下选择动作 `a` 后，未来能获得的期望累计奖励。

DQN 使用神经网络近似 Q 函数：

```text
输入：state
输出：每个 action 的 Q 值
```

### 2.4 经验回放

参考资料中的 Replay Memory 部分，说明经验回放的作用：

- 存储历史交互经验。
- 随机采样训练数据。
- 降低连续样本之间的相关性。
- 提高训练稳定性。

对应本项目文件：

- `src/replay_buffer.py`

截图建议：

- 经验回放池代码截图

### 2.5 epsilon-greedy 探索策略

说明为什么不能一开始只选 Q 值最大的动作：

- 初期网络还没学好，直接贪心容易陷入错误策略。
- epsilon-greedy 允许智能体以一定概率随机探索。
- 随着训练进行，epsilon 逐渐减小，智能体更多依赖网络决策。

### 2.6 Target Network

说明 target network 的作用：

- 使用一个相对稳定的目标网络计算训练目标。
- 避免训练目标频繁变化导致不稳定。
- 定期将当前 Q 网络参数同步到 target network。

### 2.7 改进方法

后续实验完成后补充：

- Double DQN：缓解 Q 值过估计。
- Dueling DQN：将状态价值和动作优势分开建模。
- Prioritized Experience Replay：优先学习 TD-error 较大的经验。

## 4 LunarLander 任务建模

### 4.1 任务介绍

说明 LunarLander 的目标：

智能体控制月球着陆器，通过主发动机和左右发动机调整姿态，使其尽可能平稳地降落到目标区域。

### 4.2 状态空间

LunarLander 状态空间包含 8 个连续值：

1. 横向位置
2. 纵向位置
3. 横向速度
4. 纵向速度
5. 角度
6. 角速度
7. 左支架是否接触地面
8. 右支架是否接触地面

### 4.3 动作空间

动作空间为 `Discrete(4)`：

| 动作编号 | 含义 |
|---:|---|
| 0 | 不启动发动机 |
| 1 | 启动左侧发动机 |
| 2 | 启动主发动机 |
| 3 | 启动右侧发动机 |

### 4.4 奖励机制

说明 reward 大致反映：

- 是否靠近着陆点
- 速度是否平稳
- 角度是否稳定
- 是否使用过多燃料
- 是否成功着陆或坠毁

## 5 方法设计与代码实现

### 5.1 总体流程

```text
初始化环境
初始化 DQNAgent
for 每个 episode:
    reset 环境
    while episode 未结束:
        根据 epsilon-greedy 选择动作
        环境执行动作
        存储经验
        从经验池采样并训练 Q 网络
    定期更新 target network
保存训练日志和模型
```
