# 项目结构图

以下结构按当前工程真实目录重新整理，省略 `.git`、`.venv`、`__pycache__` 和 Word 临时锁文件。

```text
LunarLander_ddqn/
├── src/                              # 核心源码
│   ├── envs.py                       # 创建 LunarLander-v3 环境
│   ├── networks.py                   # DQN 与 Dueling DQN 网络结构
│   ├── replay_buffer.py              # 普通经验回放与优先经验回放
│   ├── agent.py                      # 智能体动作选择与网络更新逻辑
│   ├── train.py                      # 训练入口、日志记录、模型保存
│   ├── evaluate.py                   # 固定 seed 的贪心策略评估
│   ├── record_video.py               # 推理 GIF、MP4 和关键帧生成
│   ├── plot_results.py               # reward/loss/epsilon 曲线绘制
│   ├── plot_last100_reward.py         # 最后 100 轮 reward 曲线
│   ├── random_agent.py               # 随机策略基线
│   └── test_env.py                   # LunarLander 环境测试
│
├── configs/                          # 实验配置
│   └── proposed.yaml                  # Proposed 方法参数配置
│
├── outputs/                          # 训练、评估和推理输出
│   ├── dqn/                          # DQN 训练日志、模型和曲线
│   ├── double_dqn/                   # Double DQN 训练日志、模型和曲线
│   ├── dueling_ddqn/                 # Dueling Double DQN 结果
│   ├── proposed/                     # Dueling DDQN + PER 结果
│   ├── proposed_alpha03/             # PER alpha=0.3 对照实验
│   ├── comparison/figures/           # 多算法 reward 对比曲线
│   ├── evaluation/                   # 20 局评估 CSV 与 JSON 汇总
│   └── videos/                       # 推理 GIF、MP4 和录制摘要
│
├── screenshots/                      # 报告截图素材
│   └── inference/                    # 随机策略与最优模型推理关键帧
│
├── report/                           # 课程报告与实验记录
│   ├── PER实验记录.md
│   ├── 评估与推理记录.md
│   ├── 实验分析记录.md
│   ├── 课程设计报告框架.md
│   └── 项目结构图.md
│
├── README.md                         # 项目说明
├── pyproject.toml                    # uv 项目依赖配置
├── uv.lock                           # uv 锁定依赖版本
└── requirements.txt                  # 课程提交用依赖清单
```
